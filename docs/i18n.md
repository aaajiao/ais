# 国际化 (i18n)

应用通过 `react-i18next` 支持中文（默认）和英文。

---

## 目录结构

```
src/locales/
├── index.ts          # i18n 配置
├── zh/               # 中文翻译
│   ├── common.json   # 共享 UI 字符串
│   ├── nav.json      # 导航
│   ├── status.json   # 版本状态
│   ├── dashboard.json
│   ├── artworks.json
│   ├── artworkDetail.json
│   ├── editions.json
│   ├── editionDetail.json
│   ├── locations.json
│   ├── chat.json
│   ├── settings.json
│   ├── import.json
│   ├── export.json
│   ├── trash.json
│   ├── history.json
│   ├── links.json
│   └── publicView.json
└── en/               # 英文翻译（相同结构）
```

---

## 使用方法

```tsx
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t } = useTranslation('namespace'); // 如 'common', 'artworks'
  return <p>{t('key.nested')}</p>;
}

// 带插值
t('selected', { count: 5 }) // "已选择 5 项" / "5 selected"
```

---

## 添加翻译

1. 在 `zh/*.json` 和 `en/*.json` 中添加键
2. 在组件中使用 `useTranslation('namespace')`
3. 用 `t('key')` 替换硬编码文本

---

## 语言切换

- 设置页面中的 `LanguageSwitcher` 组件
- 语言偏好存储在 `localStorage`（`i18nextLng`）
- 首次访问时自动检测浏览器语言

---

## 关键文件

- `src/locales/index.ts` - i18n 初始化配置
- `src/components/ui/LanguageSwitcher.tsx` - 语言切换组件
